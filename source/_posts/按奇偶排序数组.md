# 题目

给你一个整数数组 `nums`，将 `nums` 中的的所有偶数元素移动到数组的前面，后跟所有奇数元素。

返回满足此条件的 **任一数组** 作为答案。

<!--more-->

# 示例

## 示例1 

```
输入：nums = [3,1,2,4]
输出：[2,4,3,1]
解释：[4,2,3,1]、[2,4,1,3] 和 [4,2,1,3] 也会被视作正确答案。
```

## 示例2

```
输入：nums = [0]
```

# 代码

```C++
class Solution {
public:
    vector<int> sortArrayByParity(vector<int>& nums) {
        int pos = 0;
        int n = nums.size();
        for(int i = 0;i<n;i++){
            if(!(nums[i]&1)){
                swap(nums[i],nums[pos++]);
            }
        }
        return nums;
    }
};
```
