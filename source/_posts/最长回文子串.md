# 题目

给你一个字符串 `s`，找到 `s` 中最长的回文子串。

<!--more-->

# 示例

## 示例1 

```
输入：s = "babad"
输出："bab"
解释："aba" 同样是符合题意的答案。
```

## 示例2

```
输入：s = "cbbd"
输出："bb"
```

# 题目提示

- `1 <= s.length <= 1000`
- `s` 仅由数字和英文字母组成

# 思考

动态规划，`dp[i][i]`肯定是true，`dp[i][i+1]`当且仅当`s[i]==s[i+1]`

其他情况，`dp[i][j]=dp[i+1][j-1]`

# 代码

```C++
class Solution {
public:
    string longestPalindrome(string s) {
        int begin = 0;
        int ans = 1;
        int length = s.size();
        vector<vector<int>> vec(length,vector<int>(length));
        for(int i = 0;i<length;++i){
            vec[i][i] = 1;
            if(i<length-1&&s[i]==s[i+1]){
                vec[i][i+1]=1;
                ans=2;
                begin=i;
            }
        }

        for(int l = 3;l<=length;l++){
            for(int i = 0;i<length;i++){
                int j = i+l-1;
                if(j>=length) break;

                if(s[i]!=s[j]) vec[i][j]=0;
                else{
                    vec[i][j] = vec[i+1][j-1];
                }
                if(vec[i][j]&&l>ans){
                    ans = l;
                    begin = i;
                }
            }
        }

        return s.substr(begin,ans);
    }
};
```
