# 题目

两个整数之间的 汉明距离 指的是这两个数字对应二进制位不同的位置的数目。

给你两个整数 `x` 和 `y`，计算并返回它们之间的汉明距离。

<!--more-->

# 示例

## 示例1 

```
输入：x = 1, y = 4
输出：2
解释：
1   (0 0 0 1)
4   (0 1 0 0)
       ↑   ↑
上面的箭头指出了对应二进制位不同的位置。
```

## 示例2

```
输入：x = 3, y = 1
输出：1
```

# 题目提示

- `0 <= x, y <= 231 - 1`

# 思考

先让x，y异或，然后计算他们每一位为1的数量

# 代码

```C++
class Solution {
public:
    int hammingDistance(int x, int y) {
        int s = x ^ y, ret = 0;
        while (s) {
            ret += s & 1;
            s >>= 1;
        }
        return ret;
    }
};
```

# 总结

x&1 = x的最低位
