# 题目

给你一个整数 `n` ，求恰由 `n` 个节点组成且节点值从 `1` 到 `n` 互不相同的 **二叉搜索树** 有多少种？返回满足题意的二叉搜索树的种数。

# 代码

```C++
class Solution {
public:
    int numTrees(int n) {
        vector<int> ans(n+1,0);
        ans[0] = 1;
        ans[1] = 1;
        for(int i = 2;i<=n;++i){
            for(int j = 1;j<=i;++j){
                ans[i]+=ans[j-1]*ans[i-j];
            }
        }
        return ans[n];
    }
};
```
