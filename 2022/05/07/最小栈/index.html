最小栈

发表于 2022-05-07 阅读次数：
本文字数： 1.3k 阅读时长 ≈ 1 分钟

题目

设计一个支持 push ，pop ，top 操作，并能在常数时间内检索到最小元素的栈。

实现 MinStack 类:

MinStack() 初始化堆栈对象。
void push(int val) 将元素val推入堆栈。
void pop() 删除堆栈顶部的元素。
int top() 获取堆栈顶部的元素。
int getMin() 获取堆栈中的最小元素。

题目提示

-231 <= val <= 231 - 1
pop、top 和 getMin 操作总是在 非空栈 上调用
push, pop, top, and getMin最多被调用 3 * 104 次

思考

维护两个栈，一个和往常一样，另一个每次push的时候都push当前最小值和当前值的最小值
