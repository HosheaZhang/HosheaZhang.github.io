# 题目

数字 `n` 代表生成括号的对数，请你设计一个函数，用于能够生成所有可能的并且 **有效的** 括号组合。

<!--more-->

# 示例

## 示例1 

```
输入：n = 3
输出：["((()))","(()())","(())()","()(())","()()()"]
```

## 示例2

```
输入：n = 1
输出：["()"]
```

# 题目提示

- `1 <= n <= 8`

# 思考

dfs

# 代码

```C++
class Solution {
public:
    vector<string> generateParenthesis(int n) {
        vector<string> ans;
        string tmp = "";
        dfs(tmp,ans,n,0,0);
        return ans;
    }

    void dfs(string& tmp,vector<string>& ans,int n,int l,int r){
        if(tmp.size() == n*2){
            ans.push_back(tmp);
            return;
        }
        if(l<n){
            tmp.push_back('(');
            dfs(tmp,ans,n,l+1,r);
            tmp.pop_back();
        }
        if(r<l){
            tmp.push_back(')');
            dfs(tmp,ans,n,l,r+1);
            tmp.pop_back();
        }
    }

};
```
