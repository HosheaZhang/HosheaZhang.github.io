# 题目

给定一个长度为 n 的整数数组 height 。有 n 条垂线，第 i 条线的两个端点是 (i, 0) 和 (i, height[i]) 。

找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。

返回容器可以储存的最大水量。

说明：你不能倾斜容器。

<!--more-->

# 题目提示

- `n == height.length`
- `2 <= n <= 105`
- `0 <= height[i] <= 104`

# 思考

i = 0, j = size-1

S0 = （j-i）*min(height[i],height[j]);

假设i比j小，那么如果保留i,使得j--，那么S1必然小于S0，所以肯定是保留j 所以考虑使用双指针

# 代码

```C++
class Solution {
public:
    int maxArea(vector<int>& height) {
        int l = 0;
        int r = height.size()-1;
        int ans = 0;
        while(l<r){
            int tmp = min(height[l],height[r])*(r-l);;
            ans = max(ans,tmp);
            if(height[l]<=height[r]){
                l++;
            }
            else r--;
        }
        return ans;
    }
};
```
