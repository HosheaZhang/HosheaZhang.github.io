# 题目

给你一个整数数组 `nums` ，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

**子数组** 是数组中的一个连续部分。

<!--more-->

# 示例

## 示例1 

```
输入：nums = [-2,1,-3,4,-1,2,1,-5,4]
输出：6
解释：连续子数组 [4,-1,2,1] 的和最大，为 6 。
```

## 示例2

```
输入：nums = [1]
输出：1
```

## 示例3

```
输入：nums = [5,4,-1,7,8]
输出：23
```
# 题目提示

- `1 <= nums.length <= 105`
- `-104 <= nums[i] <= 104`

# 思考

# 代码

```C++
class Solution {
public:
    int maxSubArray(vector<int>& nums) {
        int res = INT_MIN;
        int ans = 0;
        for(int i = 0;i<nums.size();++i){
            if(ans<0) ans = nums[i];
            else ans+=nums[i];
            res = max(ans,res);
        }
        return res;
    }
};
```
