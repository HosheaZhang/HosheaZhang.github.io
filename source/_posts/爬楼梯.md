# 题目

假设你正在爬楼梯。需要 `n` 阶你才能到达楼顶。

每次你可以爬 `1` 或 `2` 个台阶。你有多少种不同的方法可以爬到楼顶呢？

<!--more-->

# 示例

## 示例1 

```
输入：n = 2
输出：2
解释：有两种方法可以爬到楼顶。
1. 1 阶 + 1 阶
2. 2 阶
```

## 示例2

```
输入：n = 3
输出：3
解释：有三种方法可以爬到楼顶。
1. 1 阶 + 1 阶 + 1 阶
2. 1 阶 + 2 阶
3. 2 阶 + 1 阶
```

# 题目提示

- `1 <= n <= 45`

# 代码

```C++
class Solution {
public:
    int climbStairs(int n) {
        if(n == 1) return 1;
        int fir = 1;
        int sec = 2;
        int tmp;
        for(int i = 3;i<=n;i++){
            tmp = fir;
            fir = sec;
            sec = tmp+fir;
        }
        return sec;
    }
};
```
