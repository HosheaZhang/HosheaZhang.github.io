# 题目

给你一棵二叉树的根节点 `root` ，翻转这棵二叉树，并返回其根节点。

<!--more-->

# 示例

## 示例1 

{% asset_img 1.png %}

```
输入：root = [4,2,7,1,3,6,9]
输出：[4,7,2,9,6,3,1]
```

## 示例2

```
输入：root = [2,1,3]
输出：[2,3,1]
```

## 示例3

```
输入：root = []
输出：[]
```
# 题目提示

- 树中节点数目范围在 `[0, 100]` 内
- `-100 <= Node.val <= 100`

# 思考

递归地进行转换，每一层每一层地进行转换即可

# 代码

```C++
class Solution {
public:
    TreeNode* invertTree(TreeNode* root) {
        if (root == nullptr) {
            return nullptr;
        }
        TreeNode* left = invertTree(root->left);
        TreeNode* right = invertTree(root->right);
        root->left = right;
        root->right = left;
        return root;
    }
};
```
