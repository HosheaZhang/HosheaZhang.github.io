# 题目

给你一个整数数组 `nums` ，数组中的元素 **互不相同** 。返回该数组所有可能的子集（幂集）。

解集 **不能** 包含重复的子集。你可以按 **任意顺序** 返回解集。

<!--more-->

# 示例

## 示例1 

```
输入：nums = [1,2,3]
输出：[[],[1],[2],[1,2],[3],[1,3],[2,3],[1,2,3]]
```

## 示例2

```
输入：nums = [0]
输出：[[],[0]]
```

# 题目提示

- `1 <= nums.length <= 10`
- `-10 <= nums[i] <= 10`
- `nums` 中的所有元素 **互不相同**

# 代码

```C++
class Solution {
public:
    vector<vector<int>> subsets(vector<int>& nums) {
        vector<int> tmp;
        vector<vector<int>> ans;
        ans.push_back(tmp);
        dfs(nums,tmp,ans,0);
        return ans;
    }

    void dfs(vector<int>& nums,vector<int>& tmp,vector<vector<int>>& ans,int i){
        if(i == nums.size()) return;
        tmp.push_back(nums[i]);
        ans.push_back(tmp);
        dfs(nums,tmp,ans,i+1);
        tmp.pop_back();
        dfs(nums,tmp,ans,i+1);
    }
};
```
