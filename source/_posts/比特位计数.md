# 题目

给你一个整数 `n` ，对于 `0 <= i <= n` 中的每个 `i` ，计算其二进制表示中 **1 的个数** ，返回一个长度为 `n + 1` 的数组 `ans` 作为答案。

<!--more-->

# 示例

## 示例1 

```
输入：n = 2
输出：[0,1,1]
解释：
0 --> 0
1 --> 1
2 --> 10
```

## 示例2

```
输入：n = 5
输出：[0,1,1,2,1,2]
解释：
0 --> 0
1 --> 1
2 --> 10
3 --> 11
4 --> 100
5 --> 101
```

# 题目提示

- `0 <= n <= 105`

# 思考

考虑到一个结论：`x&=(x-1)是`将二进制表示的最后一个1变成0

# 代码

```C++
class Solution {
public:
    vector<int> countBits(int n) {
        vector<int> ans(n+1);
        for(int i = 0;i<=n;++i){
            ans[i] = count(i);
        }
        return ans;
    }
    int count(int i){
        int ones = 0;
        while(i!=0){
            i&=(i-1);
            ++ones;
        }
        return ones;
    }
};
```
